# Java 独立war文件代码示例

## 项目介绍

本项目是一个简单的 Java war Servlet 示例，包含一个war文件和源码目录，用户只需要把war文件提交即可:

```
.
├── README.md
|── java-hello.war
└── src
    └── main
        ├── java
        │   └── net
        │       └── goodrain
        │           └── samples
        │               └── HelloServlet.java
        └── webapp
            ├── WEB-INF
            │   └── web.xml
            └── index.html
```

## 代码要求
- 包含至少一个 war 包
- war包需要放在根目录
